# pszponder-npm

> Get to know Piotr Szponder using your command line with Node.js

## Purpose

This is a simple npm package which will log out to the console a bio of myself.

## Usage

Run either of the two commands to run this CLI:

```bash
# Run using npm
npx pszponder
```

```bash
# Run using pnpm
pnpm dlx pszponder
```
